Политика конфиденциальности

Руководство по использованию

Сокрытие данных

  • Выберите метод: LSB для изображений или LSB для аудио.
  • Загрузите контейнер (изображение PNG/BMP/TIFF или аудио WAV).
  • Загрузите файл с данными, которые хотите скрыть.
  • Опционально укажите пароль для дополнительной защиты.
  • Нажмите «Начать сокрытие» — скачайте полученный стего-файл.

Извлечение данных

  • Загрузите стего-файл (изображение или аудио с уже скрытыми данными).
  • Если при сокрытии использовался пароль — введите его.
  • Нажмите «Извлечь данные» — скачайте восстановленный файл.

Поддерживаемые форматы

  • Контейнеры: PNG, BMP, TIFF, WAV
  • Скрытые данные: Любые файлы (до 50 МБ)

Безопасность

Все операции происходят в вашем браузере или на сервере без сохранения файлов. Пароли не хранятся — они используются только для шифрования/дешифрования в момент операции.
